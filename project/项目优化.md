### 2、为data中的常量添加Object.freeze
* vue会遍历data中的属性，为属性添加getter和setter，应尽量减少data中的属性
```js
import { tableColumns } from '@/untils/tableColumns'
...
  data () {
    return {
      tableColumns: Object.freeze(tableColumns)
    }
  }
...

// 或者在定义常量时添加
const STATES = Object.freeze({
  'SUCCESS': 1,
  'FAIL': 2,
  'WAITING': 3
})
```
